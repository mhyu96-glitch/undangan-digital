# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-04

### Added
- 🎉 Initial release of Baby Aqiqah Invitation
- 🌊 Beautiful marine/ocean theme with underwater animations
- 📱 Responsive mobile-first design
- 🎵 Background music player with controls
- 📸 Photo gallery with lightbox functionality
- 📝 Guest wishes form with real-time display
- ⏰ Countdown timer to event date
- 🗺️ Google Maps integration for event location
- 💳 Digital envelope for monetary gifts
- 🎨 Glass morphism UI effects
- ✨ Smooth page transitions and micro-interactions

### Features
- **Cover Page**: Welcome screen with baby photo and invitation details
- **Profile Tab**: Baby profile with Islamic blessing
- **Event Tab**: Schedule, countdown, and location with maps
- **Wishes Tab**: Photo gallery and guest message system
- **Thanks Tab**: Gratitude message and bank account details

### Technical
- Built with React 18 and modern hooks
- Framer Motion for smooth animations
- Tailwind CSS for styling
- Vite for fast development and building
- Custom hooks for audio, countdown, and localStorage
- Reusable UI components library
- Utility functions for formatting and validation

### Animations
- Floating sea creatures and coral reefs
- Swimming fish schools with different speeds
- Rising bubbles animation
- Gentle seaweed swaying
- Pulsing sunlight rays
- Ring pulse effects on interactive elements

### Accessibility
- Proper focus states for keyboard navigation
- Screen reader friendly markup
- Adequate color contrast ratios
- Touch-friendly tap targets for mobile
- Semantic HTML structure

### Performance
- Optimized bundle splitting
- Lazy loading for images
- Efficient animation performance
- Minimal re-renders with proper state management

### Browser Support
- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile browsers (iOS Safari, Chrome Mobile)
- Responsive design for all screen sizes
